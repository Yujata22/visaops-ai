# VisaOPS AI

VisaOPS AI is an experimental U.S. immigration scenario explorer built with Streamlit.

Users can enter fictional visa situations, organize important dates, and explore educational guidance.

## Current Features

- Visa scenario form
- Event and filing dates
- Scenario summary
- Educational-use disclaimer

## Planned Features

- Visa timeline generator
- Rule-based scenario analysis
- USCIS knowledge-base retrieval
- Source citations
- What-if simulator

## Run Locally

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
